# Real Time Video Recorder
My simple video recorder using OpenCV with real-time recording, FPS display, and image processing features.


## 📌 Description
This is a real-time video recorder built using OpenCV.

## 🎯 Features
- Camera preview
- Video recording (SPACE key)
- REC indicator with timer (HH:MM:SS)
- FPS display
- Brightness / Contrast control
- Flip mode

## ⌨️ Controls
| Key | Function |
|-----|--------|
| SPACE | Start / Stop recording |
| ESC | Exit |
| F | Flip |
| U / D | Brightness |
| C / X | Contrast |
